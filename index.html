All about Web Development

HTML

HTML stands for Hyper Text Markup Language and is the standard markup language for creating Web pages. HTML describes the structure of a Web page AND consists of a series of elements. HTML elements tell the browser how to display the content and label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

CSS

CSS stands for Cascading Style h3eets and it describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

JavaScript

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web.

GIT

Git is a version control tool (software) to track the changes in the source code. GitHub is a web-based cloud service to host your source code(Git repositories). It is a centralized system. Git doesn't require GitHub but GitHub requires Git.
